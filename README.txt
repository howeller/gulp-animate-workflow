To install the npm modules for this project follow the steps below. 
Ensure that the files "package.json" & "package-lock.json" are present in your project root directory.

1.	Open up a Terminal window and navigate to this folder:

		cd this/project/folder

2.	Run this command:

		npm install

A folder named "node_modules" should now be in this project folder and you are not ready to run the tasks that are written in gulpfile.js
